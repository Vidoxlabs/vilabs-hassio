# Vi-Memory (PostgreSQL)

A Home Assistant add-on providing PostgreSQL 16 with pgvector extension.

## Description

Transform Home Assistant's data storage layer into a powerful, vector-enabled database.

This addon provides:

- Production-grade PostgreSQL with pgvector for semantic search
- Home Assistant Recorder compatible schema
- Persistent storage for embeddings, events, and historical data
- Configuration options for performance tuning
