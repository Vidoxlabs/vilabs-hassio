# SilverBullet Add-on for Home Assistant

SilverBullet is a hackable, extensible, open-source personal knowledge management system that stores your notes as plain markdown files in your `/share` directory.

## Configuration

This add-on maps `/share/silverbullet` as the "Space" for your notes.

## Installation

1. Add this repository to your Home Assistant Add-on Store.
2. Install the **SilverBullet** add-on.
3. Start the add-on.
4. Access the web interface via the "OPEN WEB UI" button or the sidebar.
